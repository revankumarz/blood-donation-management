# Blood Donation Management System

A comprehensive system for managing blood donations, inventory, and donor records.

## About

This project aims to streamline blood bank operations by providing tools for:
- Donor registration and management
- Blood inventory tracking
- Donation scheduling
- Blood request management
- Reporting and analytics
